Clean & Corrected Notes (Short & Clear)

1. document.createElement() → Used to create a new HTML element using JavaScript

2. appendChild() → Used to add the created element inside a parent element.

Steps:

1. Create a main div in HTML and give it height and width of 100%.

2. Create a button and center it using CSS. Give it position: relative and z-index so it stays on top.

3. In JavaScript, select both the button and main.

4.Add a click event listener on the button

5. Inside the function, create a new element using document.createElement().

6. If needed, use innerHTML to add content

7. Use main.appendChild() to add it inside the main div.
